______________________________________________________________________________
                            Commands

- build:
    Reads the configuration file in given folder
    and runs the Makefile, providing the config parameters to it
    and building the target:

        make all [config parameters...]

    Arguments:

        --arch      Architecture you want to build your module for.
                    Default value: x86_64.

    Usage:

        vqbuild build [folder] [args]

- clean:
    Reads the configuration file in given folder
    and runs the Makefile, providing the config parameters to it.
    It cleans the build artifacts:

        make clean [config parameters...]

    Arguments:

        --arch      Architecture you want to clean your build artifacts in 
                    module for.
                    Default value: x86_64.

    Usage:

        vqbuild clean [folder] [args]
    
- compiledb:
    Reads the configuration file in given folder
    and runs the Makefile, providing the config parameters to it.
    It generates the compile_commands.json, using the Bear command:

        bear -- [make command, generated by vqbuild ...]

    Arguments:

        --arch      Architecture you want to generate 
                    compile_commands.json for. Default value: x86_64.

    Usage:

        vqbuild compiledb [folder] [args]
