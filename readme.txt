                            Vacui VQ

Vacui VQ (VQ - Via Quarter, it's used as prefix in private codebase) - a hybrid
operating system. Its primary feature is Abstraction Layers, that make kernel
portable:

- Architecture Abstraction Layer (AAL): Hides the details of
architecture.

- Boot Abstraction Layer (BAL): The first entry of operating system,
responsible for collecting generic boot information.

- Platform Abstraction Layer (PAL): Hides the platform-dependent code, such as
timers, interrupts, and memory.
